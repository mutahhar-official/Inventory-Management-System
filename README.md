# 📦 Inventory Management System in C

A console-based Inventory Management System built in C, designed to help small businesses and stores efficiently track, manage, and report on their product inventory in real time.

> _This system is built as a part of first semester Computer Programming Project by the students of NED University of Engineering and Technology, Karachi, Pakistan._

## ⚙️ System Options

| Option | Description |
|--------|-------------|
| **Add Product** | Add a new product with its name, price, and quantity to the inventory. |
| **Delete Product** | Remove an existing product from the inventory by its ID. |
| **Display Products** | View the complete list of all products currently in the inventory. |
| **Sell Product** | Process a sale, deduct stock automatically, and generate a bill for the customer. |
| **Update Product** | Modify the details of an existing product such as price or quantity. |
| **Low Stock Report** | Display a report of all products whose quantity has fallen below the minimum threshold. |
| **Save Inventory** | Write the current inventory data to a file for persistent storage. |
| **Load Inventory** | Read and restore previously saved inventory data from a file. |
| **Exit** | Safely close and exit the inventory management system. |

---

## ✨ Features

- **Menu-Driven Interface** — Simple, numbered menu for easy navigation.
- **Product Management** — Full support for adding, updating, selliing and deleting products.
- **Low Stock Alerts** — Instantly identify products running low on stock.
- **File Handling** — Save and load inventory data to ensure nothing is lost between sessions.
- **Modular Code Design** — Clean, function-based structure for readability and maintainability.
